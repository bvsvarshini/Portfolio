## 🎯 Objective

To design and develop a **personal portfolio website** that showcases my **skills, projects, experience, and achievements** in a visually appealing and interactive way.

---

## 📚 Project Overview

This is a fully responsive **personal portfolio website** built using **HTML and CSS**, featuring:

- Modern UI with gradient animations  
- Interactive sections for projects, skills, and experience  
- Smooth scrolling navigation  
- Resume integration  
- Contact information for professional reach
